# 🧾 Invoice AI Management System

A **premium, AI-powered Invoice Management Web Application** built with a modern tech stack and designed with a **luxury black & gold UI**.
This application allows users to **upload, process, fix, and generate invoices** seamlessly using OCR and AI-based logic — all running fully on the frontend.

## ✨ Features

### 🔐 Authentication
* **Admin Login:** `admin@invoice.ai` / `admin123`
* Simulated real-world email login support
* Glassmorphism UI with error validation

### 📤 Upload Invoice (AI Processing)
* Support for **PDF, JPG, PNG**
* Built-in OCR using **Tesseract.js**
* Deep extraction of Header, Date, Vendor, and Line Items

### 🧠 Smart AI Logic
* **Auto Fix:** Neural pipeline to automatically fill missing fields and fix formats.
* **Manual Fix:** Pro-grade spreadsheet editor with visual anomaly flagging.

### 📥 PDF Export
* Professional PDF generation using **jsPDF**
* Structured layouts ready for financial auditing.

### 🌓 Theme Customization
* **Dark Mode:** Luxury Black + Gold theme (Default)
* **Light Mode:** High-contrast professional theme

## 📂 Installation

```bash
# Clone the repository
git clone https://github.com/Sanjaykgcas/invoice-ai-management.git

# Navigate into project
cd smart-invoice-automation

# Install dependencies
npm install

# Run development server
npm run dev
```
